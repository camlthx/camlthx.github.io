对于普通莫队，块长设为 [$\sqrt n$](http://www.luogu.com.cn/record/99245948 ) 即可。但对于带修莫队，最好设置为 [$n^{\frac{2}{3}}$](https://www.luogu.com.cn/record/99246691) 。效果极为显著,快了将近2s。

upd 2022.11.12 

逻辑运算 ‘！’的优先级是高于位运算的，就是这么个小东西我调了一上午（哭）。

形如：
```cpp
	if(!exp)
```

最好写作：

```cpp
	if(!(exp))
```
